NTFL Public Build

Upload these files to GitHub Pages:
- index.html
- 404.html
- assets/
- data/site-data.json

How edits work:
1. Open Admin.
2. Change games, teams, rankings, awards, or history.
3. Click Download data.json.
4. Replace data/site-data.json in GitHub.
5. Commit changes and refresh the site.

Notes:
- The site uses hash routing, so team pages and other pages work on GitHub Pages.
- PPG, standings, and rankings display update from the shared data file.
