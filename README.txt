# NTFL Rebuild

Static NTFL league hub with a more modern, card-based layout.

Includes:
- Home page with featured cards
- Team pages
- Schedule
- Standings
- Rankings
- Awards
- History
- Hall of Fame
- Admin dashboard with demo login
- Drag/reorder rankings editor
- Live / upcoming / final game states
- Auto-updating PPG and standings from scores

## How to use
1. Open `index.html` in a browser or upload the folder to GitHub Pages.
2. Use the Admin page to edit data.
3. Save locally for testing, then export JSON and replace `data/site-data.json` so other visitors see the update.

## Logo
The site uses `assets/IMG_5900.png` as the primary logo and falls back to the built-in styling if needed.

## Public sharing
If you want changes to appear for everyone, upload the updated `data/site-data.json` and any changed assets back to the live host or GitHub repository.
